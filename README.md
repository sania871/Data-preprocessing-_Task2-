# Data-preprocessing-


Student Performance Analysis with NumPy & Pandas
📌 Overview
This project demonstrates how to clean, preprocess, and analyze a student dataset using Python libraries NumPy and Pandas.
It covers numerical operations, data exploration, preprocessing, and analysis to extract meaningful insights from student performance data.

🛠️ Technologies Used
Python 3.x

NumPy

Pandas

📂 Project Structure
The project is divided into four main parts:

Part 1 — NumPy Operations
Create a NumPy array from the math_score column.

Compute mean, median, maximum, and minimum values.

Normalize scores using NumPy.

Part 2 — Pandas Exploration
Load dataset using Pandas.

Display the first 5 rows.

Check data types of all columns.

Find missing values in each column.

Identify students with attendance < 70%.

Part 3 — Data Preprocessing
Handle missing values.

Convert incorrect data formats.

Convert exam_date column into datetime format.

Detect and handle outliers in math_score and science_score.

Remove duplicate rows.

Part 4 — Data Analysis
Create a new column average_score.

Find top 5 students based on average score.

Calculate correlation between attendance and marks.

Group students by gender and calculate average marks
